LandR Overview

LandR is a collection of SpaDES modules aimed at simulating forest dynamics across large spatial scales, while taking into account various disturbances that affect them (e.g., wildfire and climate change), as well as interactions with other components of forest systems, such as bird communities and ungulates (via changes in the habitat of these species), and carbon cycling.

In-depth information on the functioning, structure, and science behind LandR can be found in its manual. Similar to this website, it is a living document that is continually updated as the model evolves.

Modules

Several modules can be used to run LandR depending on the user’s needs. For LandRCBM, five different modules are frequently used. Here is a brief description of each one.

Biomass_borealDataPrep

This module prepares all necessary inputs for Biomass_core based on data available for forests across Canada forests, but focused on Western Canada boreal forest systems. This module cleans up raw input data, calculated species biomass per pixel, prepares various species traits, defines ecolocations, and creates the initial landscape conditions. For more information on the background, functioning, and potential module linkages in this module, see its manual chapter.

Biomass_speciesFactorial

This module creates and runs a factorial experiment for LANDIS-II-like species traits used in Biomass_speciesParameters. While this module does not have a manual chapter, more information along with its code can be found in its module repository.

Biomass_speciesParameters

This module calibrates species growth and mortality trait values used in Biomass_core and identifies the combination of trait values that allows a better match to the observed growth curves by matching theoretical species’ curves obtained with different trait values against observed growth curves derived from Permanent Sample Plots across Canada. More information on the functioning and specific traits calibrated in this module can be found in its manual chapter.

Biomass_regeneration

This module simulated post-fire regeneration for Biomass_core. It is mostly based on the post-disturbance regeneration mechanisms present in LANDIS-II Biomass Succession Extension v.3.2.1. This module activates post-fire mortality and activates serotiny and resprouting mechanisms for each species on affected pixels depending on their traits. Mortality behaves in a stand-replacing fashion in this module. While this module does not yet have a manual chapter, more information on its functioning, along with some of its differences with the LANDIS-II Biomass Succession Extension v.3.2.1 can be found in the README.md and Biomass_regeneration.Rmd files in its GitHub repository.

Biomass_core

This module is the core forest succession simulation module of LandR. It simulates tree cohort ageing, growth, mortality and competition for light resources, as well as seed dispersal, in a spatially explicit manner and using a yearly time step. It is based on the LANDIS-II Biomass Succession Extension v.3.2.1 with a few changes. You can find details of the differences between the two on this page. Biomass_core is intended to be used with data/calibration modules, disturbance modules and validation modules, amongst others. You can view a visual map of many modules in the SpaDES ecosystem of models and how they connect to Biomass_core here. More in-depth information on Biomass_core can be found in its manual chapter.
